# Редизайн лендинга MarkVision AI

Жёлтый на белом плохо читается и выглядит «инфобизово» — не подходит для медицинской ниши. Делаю серьёзный редизайн под доверие, как у современных клиник (Mayo, DocDoc, Doc+).

## 1. Палитра (вместо жёлтого)

- **Primary** — медицинский teal `#0F766E` (CTA, акценты, иконки)
- **Primary soft** — `#CCFBF1` (плашки success, фон цитат)
- **Accent для цифр/важного** — глубокий emerald `#059669`
- **Тревожный** (только для «ДО» и проблем) — приглушённый rose `#E11D48`, фон `#FFF1F2`
- **Нейтральные** — белый `#FFFFFF`, фон секций `#F8FAFC` (slate-50), границы `#E2E8F0`
- **Текст** — `#0F172A` (заголовки), `#475569` (основной), `#64748B` (вторичный)
- **WhatsApp CTA** — фирменный `#25D366`

Никакого жёлтого, никакого чистого чёрного.

## 2. Типографика (новые шрифты, без uppercase)

- **Заголовки**: **Manrope** 600/700 — современный, спокойный, медицинский. Без `uppercase`, без `Oswald`.
- **Body**: **Inter** 400/500 — стандарт читаемости.
- Иерархия:
  - H1: 36/44px моб, 56/64px десктоп, tracking -0.02em, weight 700
  - H2: 28/36px моб, 40/48px десктоп, weight 700
  - H3: 20/28px, weight 600
  - Body: 16/26px моб, 18/30px десктоп
- Никаких `text-7xl`, `text-6xl uppercase`, `WebkitTextStroke` — всё убираю.
- Цифры (30 млн ₸, 4 990 ₸) — крупным Manrope с цветом emerald, без обводок.

## 3. Что меняется по блокам

1. **Hero** — белый фон, slate-50 «дымка» сверху. H1 в графите, ключевые слова в teal. 3 буллета с иконками `Check` в teal-кружке. CTA-кнопка teal (primary) + secondary outline «Узнать как работает». Картинка клиники в скруглённом контейнере с мягкой тенью, без жёлтой подсветки.
2. **Кейс Бакыта** — карточка на slate-50, фото 96px круглое, «13 млн ₸» — emerald цифра без жёлтой заливки.
3. **ДО** — на rose-50 фоне, иконки `XCircle` в rose-500. Без чёрного фона.
4. **ПОСЛЕ** — на teal-50 (`#CCFBF1`) фоне, иконки `CheckCircle2` в teal-700. «30 млн ₸» emerald крупным. Без жёлтого блока.
5. **Триггер** — узкий центрированный белый блок, «3 шага» в teal-плашке.
6. **Эксперт Юрий** — карточка на белом, фото в teal-кольце, без чёрной рамки.
7. **4 проблемы** — карточки на белом с slate-200 рамкой, номер в teal-кружке.
8. **Шаги 1–3** — большой номер в teal без обводки, заголовок Manrope 700 normal-case, плашка «Пример» с border-left teal на slate-50, плашка «Результат» на teal-50.
9. **Стоимость** — крупная карточка с teal-рамкой (не жёлтая), «4 990 ₸» emerald 64px.
10. **Бонусы** — карточки на slate-50, «бесплатно» emerald, итоговая плашка teal-50 с teal текстом.
11. **Честно** — вопросы как список с teal-бордером слева. Итоговая плашка — graphite `#0F172A` (НЕ чёрный) с белым текстом и emerald цифрой.
12. **Гарантия** — на slate-50, иконка-щит в teal-круге с мягким glow.
13. **FAQ** — рамки slate-200, hover teal, шеврон teal.
14. **CTA-секции** — на slate-50, кнопка WhatsApp зелёная + микро-trust под кнопкой («2 минуты», «без обязательств»).
15. **Footer** — на slate-900 (тёмный графит ОК для футера, но без чистого `#000`), нейтральный.

## 4. UX и функционал (готовность к рекламе)

- **Sticky mobile CTA-bar** внизу экрана (WhatsApp + цена 4 990 ₸) — критично для конверсии с рекламы.
- **Smooth scroll** на якорь «#cta» с hero-кнопки «Узнать, как работает».
- **Trust-строка** под hero: «20+ клиник в РК · 85% доходимость · возврат 100%» с маленькими иконками.
- **Счётчик мест** в блоке цены: «Осталось 7 из 10 мест» (статично, но визуально якорит дефицит).
- **Микро-анимации**: fade-in-up на скролле через CSS `@starting-style` или простой IntersectionObserver-хук (без тяжёлых либ).
- **Hover-состояния** на всех CTA и карточках.
- **Кликабельный телефон/WhatsApp** в футере (`tel:` и `wa.me`).
- **Aria-labels** на кнопках-иконках, alt на изображениях.

## 5. SEO и трекинг (для рекламы)

- `<title>` ≤ 60 симв: «Поток пациентов на дорогие услуги — MarkVision AI»
- `meta description` ≤ 155 симв
- `og:title`, `og:description`, `og:image` (hero.jpg), `twitter:card`
- `lang="ru"` на html
- `<link rel="canonical">`
- JSON-LD `Organization` + `Service`
- Подготовлены места для пикселей: `<head>` слот для Meta Pixel, Google Tag, Яндекс.Метрика — добавлю `<script>` плейсхолдеры с комментариями «вставить ID»
- Кнопки WhatsApp — `data-event="cta_whatsapp_click"` для трекинга

## 6. Технические изменения

- `src/styles.css`: убираю Oswald, добавляю Manrope. Меняю токены в `:root` (oklch) на teal-палитру.
- `src/routes/__root.tsx`: добавляю `<html lang="ru">`, `<link rel="canonical">`, скриптовые плейсхолдеры пикселей в `<head>` через `Scripts`.
- `src/routes/index.tsx`: переписываю все 19 блоков под новую палитру, убираю весь uppercase кроме мелких бейджей, убираю `text-7xl`/`text-6xl`, убираю жёлтый.
- Новые мелкие компоненты (в том же файле для простоты): `TrustBar`, `StickyMobileCta`, `Reveal` (fade-in на скролле).
- Добавляю `IntersectionObserver`-хук в `src/hooks/use-reveal.ts`.

## 7. Что НЕ делаю

- Не меняю тексты (контент остаётся).
- Не добавляю формы/бэкенд — всё в WhatsApp.
- Не подключаю внешние библиотеки анимаций (только CSS + IO).
- Не трогаю Cloud/БД — лендинг полностью статический.

После одобрения — реализую за один проход, затем сниму скриншот и проверю всю страницу на десктопе и мобайле.
