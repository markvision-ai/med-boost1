# План: убрать маркетинговый сленг + добавить кейс «Алматы»

## 1. Чистка текстов под владельца медцентра

Меняю формулировки, которые звучат «для маркетологов», на прямое обращение к владельцу клиники.

| Сейчас | Станет |
|---|---|
| Система привлечения клиентов (СПК) | Система привлечения пациентов (СПП) |
| Кейс клиента | Кейс клиники |
| Клиент MarkVision AI | Клиника-партнёр MarkVision AI |
| «не получить заявку, а продать услугу» | «не просто принять звонок, а записать пациента на лечение» |
| «Администратор, отвечающий на заявку за 5 минут» | «Администратор, который перезванивает за 5 минут» |
| «как привлекать от 5 пациентов в день через СПК» | «как ежедневно получать 5+ пациентов на дорогостоящие услуги» |
| «85% загрузки кабинетов благодаря СПК» | «85% загрузки кабинетов каждый день» |
| «реклама, которая не привела пациентов» | оставляю — это про опыт владельца |
| meta description «СПК и удержание» | «адресное предложение, поток пациентов и их удержание» |

Все упоминания «СПК» → «СПП» по тексту (4 места).

## 2. Новый блок «Результаты из реальных клиник»

Вставляю **после** Кейса Бакыта (раздел 01) и **перед** «До и после» (раздел 02). Новый Chapter — `01.5` / «Кейс клиники».

Структура:
- Eyebrow: «Реальные результаты»
- H2: «Результаты из реальных клиник»
- Подпись: «Разные города, разные ниши — одна система.»
- Карточка-история «Клиника из Алматы»:
  - **Было** (rose-плашка): «Один администратор не справлялся со звонками. Записывал всех подряд — приходило только 30% от записанных.»
  - **Что сделали** (teal-плашка, чек-лист):
    - Вывели 2 сотрудниц в отдельный колл-центр
    - Обучили скриптам продажи первичной консультации
    - Поставили задачу: записать с предоплатой или полной оплатой
- **Карточка-результат** (как на скриншоте, но в teal-палитре):
  - Eyebrow-чип «РЕЗУЛЬТАТ ЗА 2 НЕДЕЛИ» (тёмно-teal фон, emerald текст, иконка TrendingUp)
  - 4 квадратные мини-карты: иконка + крупное число + подпись
    1. 📢 `830 000 ₸` — Расходы на рекламу
    2. 📥 `415` — Обращений
    3. 🩺 `83` — Платных диагностик
    4. 👤+ `29` — Новых пациентов
  - Полоса: 🧾 «Средний чек: **350 000 ₸**»
  - Большая итоговая плашка: 💰 `+13 000 000 ₸` emerald 56-72px + «выручки в кассу»

Цвета — **существующая teal-палитра**, не синий из скрина. Фон карточки `bg-[var(--primary)]` (teal-700), текст белый, акцентные цифры — `var(--emerald)`, мини-карты — `bg-white/10` с `border-white/15`.

Иконки (lucide): Megaphone, Inbox, Stethoscope, UserPlus, Receipt, Wallet, TrendingUp.
