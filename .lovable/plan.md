# VSL-лендинг MarkVision AI (светлая тема, медицинская)

Одностраничный лендинг на React + Tailwind по структуре RichDent, адаптированной под медицинские центры. Все тексты из брифа — без изменений. Визуально — чистая, светлая, "клиническая" эстетика, внушающая доверие.

## Дизайн (светлая медицинская тема)

- **Фон:** белый `#FFFFFF` и мягкий холодный `#F4F8FB` для чередования секций
- **Текст:** глубокий графит `#0F172A` (заголовки), `#475569` (основной)
- **Основной акцент:** медицинский бирюзово-синий `#0EA5A4` / `#0284C7` (CTA, заголовки шагов, иконки)
- **Дополнительный акцент:** мягкий мятный `#E0F7F5` для фонов карточек "ПОСЛЕ"/успеха
- **Тревожный акцент:** приглушённый коралл `#EF6E6E` для блоков "ДО"/проблем и цифры "30 млн ₸"
- **Типографика:** Inter / "Manrope" для заголовков (жирный, но не uppercase, без агрессии), Inter для текста — спокойно, по-медицински
- **Карточки:** белые на сером фоне, тонкая граница `#E2E8F0`, мягкая тень, скругление `rounded-2xl`
- **Кнопки WhatsApp:** фирменный зелёный `#25D366` с белым текстом, крупные, со скруглением
- **Иконки:** Lucide (Stethoscope, ShieldCheck, HeartPulse, CheckCircle2, AlertCircle) — line-style, в цвет акцента
- **Mobile-first**, max-w-3xl по центру, щедрые отступы (py-16..24)

## Технические детали

- Vite + React + Tailwind (текущий шаблон)
- Дизайн-токены через HSL-переменные в `src/index.css` (`--primary`, `--accent-coral`, `--accent-mint`, и т.д.), используются Tailwind-классы через `tailwind.config.ts`
- Структура: `src/pages/Index.tsx` собирает секции из `src/components/sections/*` (Hero, CaseStudy, BeforeAfter, ExpertStory, Problems, StepOne, StepTwo, StepThree, Outcome, Pricing, Bonuses, HonestTalk, Cta, Guarantee, Faq, Footer)
- Кнопки WhatsApp → `https://api.whatsapp.com/send/?phone=+77472842595&text=%D0%9C%D0%B5%D0%B4%D1%86%D0%B5%D0%BD%D1%82%D1%80`, `target="_blank" rel="noopener"`
- FAQ — нативный `<details>/<summary>` со стилизацией
- SEO: `<title>MarkVision AI — система роста для медицинских центров</title>`, meta description под 160 символов, единственный H1, semantic HTML (`<section>`, `<article>`)
- Иконки — `lucide-react` (уже есть в шаблоне shadcn)
- 2–3 ключевых изображения через imagegen (premium для портретов): портрет владельца клиники, портрет эксперта, иконка-щит гарантии. Остальное — иконки Lucide
- Адаптивно: одна колонка на мобайле, 2 колонки для ДО/ПОСЛЕ и бонусов на ≥md
